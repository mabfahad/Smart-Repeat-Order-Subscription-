Smart Repeat Order is a WooCommerce extension that allows customers to easily schedule repeat orders for their favorite products.

== Description ==
Smart Repeat Order helps customers reorder frequently purchased items on a weekly, bi-weekly, or monthly basis. Admins can manage repeat orders and send email reminders.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the "Repeat Order" button in the My Account page to reorder items.

== Frequently Asked Questions ==
= Can I schedule orders automatically? =
No, this plugin only allows repeating orders manually via the "Repeat Order" button.
